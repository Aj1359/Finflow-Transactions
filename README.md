# 💎 FinFlow Enterprise — Strategic AI Dashboard

**FinFlow Enterprise** is a high-fidelity, cyber-executive financial management system designed for modern wealth management. It combines **Gemini-powered financial intelligence, 3D dynamic visualizations, and a premium multi-aesthetic UI** to deliver a seamless, strategic tracking experience.

---

## 🚀 Enterprise Highlights

- **Executive Branding**: Royal Purple & Gold prestige (Light) / Neon-Cyber intensity (Dark).
- **FinBot AI Intelligence**: Autonomous transaction logging and 50/30/20 rule analysis via Gemini Pro 1.5.
- **3D Visualization Suite**: Isometric card projections, cylindrical depth gradients, and luxury entry animations.
- **Capital Allocation**: Admin-only budget controls with proactive risk alerts and balance-checked enforcement.
- **High-Performance Architecture**: Vite-powered React 18 with high-precision state management.

---

## ✨ Advanced Features

### 🎨 Strategic UI/UX
- **Interactive 3D Projections**: Cards tilt and react to isometric perspective on hover.
- **Cylindrical Data Geometry**: Charts utilize multi-stop gradients to simulate 3D volume.
- **Adaptive Precision**: Seamless transition between high-contrast "Executive" and "Neon" aesthetics.

### 🧠 FinBot — AI Intelligence
- **Contextual Intent Detection**: Automatically differentiates between logging, analysis, and navigation.
- **Budget Enforcement**: Proactive warnings when transactions approach or exceed monthly category limits.
- **Natural Language Command**: "Add ₹1200 for Client Lunch" -> Logged and validated in real-time.

### 🛡️ Operational Control
- **Role-Based Access**: Specialized views and controls for Admin vs. Viewer roles.
- **Liquidity Tracking**: Real-time remaining liquidity calculations across all budget headers.

---

## 🛠️ Technology Stack

- **Frontend**: React 18, Vite
- **AI Engine**: Google Gemini Pro 1.5
- **Visuals**: Chart.js 4.x with custom 3D logic
- **Icons**: Lucide 
- **Persistence**: LocalStorage (Offline-First)

## 📦 Getting Started

### 1️⃣ Clone & Install
```bash
git clone https://github.com/Aj1359/Finflow-Transactions.git
cd Finflow-Transactions
npm install
```

### 2️⃣ Configure Intelligence
Create a `.env` file and add your **Gemini API Key**:
```env
VITE_GEMINI_API_KEY=your_gemini_api_key_here
```

### 3️⃣ Launch
```bash
npm run dev
```
